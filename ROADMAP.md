# Sentinel CI Roadmap 🗺️

**Sentinel CI** (AI Code Guardian engine) has completed its initial MVP and is now in **Beta (V1.0)**. We are transitioning from a simple GitHub Action into a full **SaaS-based security and architecture platform**.

---

## ✅ Released: V1.0 Beta (Current)
*   **Multi-LLM Orchestration**: Intelligent failover between OpenAI, Anthropic, and Gemini.
*   **Repo-Wide Context Awareness**: Automatic detection of tech stacks and project-specific dependencies.
*   **Architectural Guardrails**: 
    - **Intelligent (AI)**: Natural language practices/guidelines.
    - **Deterministic (Regex)**: Hard laws for imports, naming, and structure.
*   **Industrial Standards**: Full mapping to OWASP Top 10 and CWE.

---

## 🚀 Q2 2026: The SaaS Evolution
We are moving beyond individual repositories to a centralized management interface.
- **Central Web Dashboard**: A unified UI to view scan results across all your repositories.
- **Native GitHub App**: Transitioning from a classic Action to a native GitHub App for better permissions and faster integration.
- **Cross-Repo Trends**: Analyze security posture and technical debt over time across your entire organization.
- **High-Performance Caching**: Intelligent result caching to minimize AI token usage for recurring code patterns.

---

## 🛠️ Q3 2026: Intelligent Autonomy
- **AI Auto-Remediation (Auto-PRs)**: The platform won't just find bugs—it will generate the fix and open a Pull Request for you.
- **Custom Rule Builder (Drag-and-Drop)**: A visual interface for building deterministic "Hard Laws" without writing regex.
- **Slack & Jira Integrations**: Real-time alerts and automated ticket creation for critical security findings.

---

## 🏢 Q4 2026: Enterprise Governance
- **SSO/SAML Support**: Enterprise-grade identity management.
- **Compliance Export**: One-click reports for SOC2, ISO27001, and HIPAA compliance.
- **Self-Hosted Runner**: For high-compliance environments that require total data isolation.

---

## 📈 Project Lifecycle

1. **V1.0 Beta**: Core Engine + Multi-LLM (Current).
2. **V2.0 SaaS**: Central Dashboard & Organization-wide Analytics.
3. **V3.0 Autonomy**: Automated Remediation & Workflow Integrations.
4. **V4.0 Enterprise**: Governance, Compliance, and Self-Hosted options.
